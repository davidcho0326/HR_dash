# Claude Code 세션 요약 - HR_dash 프로젝트

## 프로젝트 개요
- **프로젝트**: AI 기반 HR 관리 대시보드
- **기술 스택**: Vite + React + TypeScript
- **배포**: Vercel (https://hr-dash-xi.vercel.app/)

## 최근 작업 내용 (2025-01-09)

### 1. 팀 구성 승인 → 전체 앱 동기화 기능
- **State Lifting 패턴** 적용: `OrchestratorApp`에서 전역 상태 관리
- `useState`로 `projects`, `employees`, `workModules` 관리
- 모든 View 컴포넌트(Dashboard, Projects, Talent, Analytics)에 props로 전달

### 2. Gemini API 확장
- **파일**: `src/services/gemini.ts`
- **모델**: `gemini-3-flash-preview`로 업그레이드
- `WorkModule` 타입 추가 (기술 스택, 담당자, 예상 시간)
- AI가 팀 구성 시 Work Modules도 함께 생성

### 3. 주요 수정 파일
- `src/app/App.tsx`: 전역 상태 관리, `addProject` 함수, `handleApproveTeam`
- `src/services/gemini.ts`: Gemini API 통합, WorkModule 생성

## 다음 작업자를 위한 컨텍스트

### 핵심 아키텍처
```
OrchestratorApp (App.tsx)
├── useState: projects, employees, workModules
├── addProject(): 새 프로젝트 + WorkModules 추가
└── Views (props로 데이터 전달)
    ├── DashboardView
    ├── ProjectView (AI 팀 구성 + 승인)
    ├── TalentView
    └── AnalyticsView
```

### 환경 변수
- `VITE_GEMINI_API_KEY`: Gemini API 키 (`.env.local`에 설정)

### 테스트 방법
1. `npm run dev` → http://localhost:5173
2. Projects 탭 → AI 팀 구성 요청 → 팀 구성 승인
3. Dashboard, Analytics에서 새 프로젝트 확인
